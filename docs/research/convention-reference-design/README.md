# Convention Reference Design Research

Evidence base for redesigning the post-practice reference page that appears below the guided convention flow. The current reference is comprehensive but obtuse; this research grounds a redesign in documentation-design, learning-science, and game-reference-exemplar evidence.

## Files

- `evidence-summary.md` — synthesis of three parallel research waves (reference-manual design, learning science, bridge/game exemplars) into a single set of actionable principles
- `reference-page-template.md` — concrete template for every convention reference page: section order, visual treatments, consistency rules, what is deliberately omitted
- `multi-bid-presentation/` — follow-up research pass on conventions with multiple peer entry-point bids (Jacoby transfers, Bergen, DONT, Two-way NMF, Unusual NT). Start with `evidence-map.md`; `question.md`, `seed-papers.md`, and `evaluation.md` are the supporting artifacts.

Raw per-wave reports for the original pass live at `_output/contexts/260412-0935-research-reference-manuals-best-research-regarding-r/notes/`. Raw per-wave reports for the multi-bid pass (landscape-keyword/semantic/reviews, expansion-backward/forward/crossdomain/adversarial) live at `_output/research/20260414-210905-bridge-multi-bid-conventions/`.

## Use This When

- Redesigning the reference section on a convention learn/practice page
- Adding a new convention and deciding what the reference page should contain
- Writing or reviewing convention authoring guidance (`docs/guides/convention-authoring.md`)
- Weighing a tradeoff between density and teaching (this research resolves most of them)

## Framing Constraint

The reference page must serve **both** a first-time learner (arriving directly from the guided flow) **and** a returning intermediate (3+ months later, glancing during practice). This is the central design tension — known in the literature as the **expertise reversal effect** — and every recommendation here is calibrated to serve both through layered, progressive disclosure rather than two separate artifacts.
