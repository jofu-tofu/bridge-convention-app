# Bridge Practice App

App for drilling bridge bidding conventions (Stayman, Gerber, DONT, Bergen Raises). Built with Svelte 5 + TypeScript + Rust/WASM.

## Getting Started

```bash
# Install all dependencies
npm install

# Start the dev server
npm run dev

# Run all tests
npm run test:all
```

## Recommended IDE Setup

[VS Code](https://code.visualstudio.com/) + [Svelte](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode) + [rust-analyzer](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer).
