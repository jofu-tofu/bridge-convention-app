# Bridge Practice App

Desktop app for drilling bridge bidding conventions (Stayman, Gerber, DONT, Bergen Raises). Built with Tauri 2.0 + Svelte 5 + TypeScript.

## Getting Started

```bash
# Install all dependencies
npm install

# Start the dev server
npm run dev

# Run all tests
npm run test:all
```
